# Setup
```sh
yarn
yarn dev
```
[open browser](http://localhost:3000)


# Local development

in a seperate terminal:
```sh
npx netlify-cms-proxy-server
```

Start the website:
```sh
yarn run dev
```

Run the admin
```
http://localhost:3000/admin
```
or
```sh
cd public/admin
python -m http.server
```
